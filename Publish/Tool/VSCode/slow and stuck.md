**Force quit an app:**  
Press `Option + Command + Esc` → select the app → click **Force Quit**.

**Open VS Code without extensions:**  
Run this command in terminal:  
`code --disable-extensions`

**Show running VS Code extensions:**  
Press `Cmd + Shift + P` → type `Developer: Show Running Extensions` → press **Enter**.
