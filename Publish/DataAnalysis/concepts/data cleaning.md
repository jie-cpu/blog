# Data Cleaning

Dealing with "dirty" data — data that is incomplete, noisy, inconsistent, or intentionally incorrect.

## Types of Dirty Data

| Type | Description |
|------|-------------|
| **Incomplete** | Missing data values |
| **Noisy** | Errors, outliers, random fluctuations |
| **Inconsistent** | Contradictory values or formatting issues |
| **Intentional** | Deliberately wrong entries (e.g., fake info) |

## Handling Methods

| Problem | Solution |
|---------|----------|
| Missing entries | **Imputation** (fill in missing values) |
| Incorrect entries | **Inconsistency detection** (find and correct errors) |

## Common Imputation Techniques

- Mean/median/mode imputation
- Forward/backward fill (time series)
- KNN imputation
- Regression imputation

## One-Line Summary

> Data cleaning = fixing incomplete + noisy + inconsistent data before analysis.
