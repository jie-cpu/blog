# Summary 

Q-Learning is a reinforcement learning algorithm that helps an [[agent]] learn to make decisions. Key points include:
1. **Goal**: Maximize rewards by learning the best actions to take in different situations.
2. **Q-Values**: Stores expected future rewards for actions in a Q-table.
3. **Learning**: The agent explores the environment, receives rewards, and updates Q-values based on feedback.
4. **Exploration vs. Exploitation**: Balances trying new actions and using known rewarding actions.
5. **Convergence**: Q-values improve over time, leading to optimal decision-making.
  Q-Learning is used in various fields, such as robotics and game playing.
