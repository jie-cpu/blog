# Summary 

In machine learning models (especially in linear regression and neural networks), **Weights** are the **trainable parameters** that determine the strength or importance of each input feature in influencing the model's output.

Their core purpose is to **adjust the contribution of each piece of input data** as it flows through the model, ultimately shaping the model's predictions.

- **Adjustable Values:** Weights are numerical values (often initially random) that the learning algorithm (like Gradient Descent) continuously tweaks during training.
- **Feature Importance:** A larger absolute weight value for a particular input feature means that feature has a stronger influence on the prediction. A weight close to zero means that feature has little influence.
- **Mathematical Operation:** In models like linear regression, weights are multiplied by the input features, and these products are summed up (along with a bias term) to form the final prediction. In neural networks, weights connect neurons between layers.
In essence, weights are the "knobs" that the machine learning algorithm turns during training. By adjusting these knobs, the model learns which parts of the input data are most relevant and how they should combine to make accurate predictions.
