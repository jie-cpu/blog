# Summary 

**Image Classification** is a fundamental task in **computer vision** (a field of AI), which is a type of **supervised learning**.

Its core purpose is to **assign a predefined category or label to an entire input image**.

- **Input:** An image (e.g., a photo of a cat, a handwritten digit, a landscape).
- **Process:** A machine learning model (often a Convolutional Neural Network) analyzes the visual content within the image. It learns to identify patterns, features, and textures that are characteristic of different categories from a large dataset of labeled images.
- **Output:** A single categorical label that best describes the main subject or content of the image (e.g., "cat," "number 7," "forest").
In essence, image classification is like teaching a computer to look at a picture and tell you what's in it, choosing from a list of things it already knows about.
