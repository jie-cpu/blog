# Summary 

In reinforcement learning, a **policy** is a plan that tells an [[agent]]what action to take in different situations (states).
1. **Definition**: A policy can be:
    - **Deterministic**: Always chooses the same action for a state.
    - **Stochastic**: Chooses actions based on probabilities.
2. **Purpose**: It helps the agent decide how to act to get the most rewards over time.
3. **Notation**: Policies are often written as π, where π(s) shows the action for state s.
