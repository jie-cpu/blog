# Summary 

**Unsupervised Learning** is a type of machine learning where the model learns from **unlabeled data**, without any explicit guidance or "correct answers."

Its core purpose is to **discover hidden patterns, structures, or relationships within the data itself**.

- **Unlabeled Data:** The dataset consists only of input examples, with no predefined output labels.
- **Pattern Discovery:** The model explores the data to find inherent groupings, unusual points, or simplified representations on its own.
- **No "Right" Answer:** Unlike supervised learning, there's no direct feedback on whether the model's findings are "correct" or "incorrect" in the traditional sense, as there are no labels to compare against.
In essence, unsupervised learning is like giving a child a pile of mixed toys and asking them to sort them into groups that make sense to them, based on similarities they observe, without telling them what the groups should be.
