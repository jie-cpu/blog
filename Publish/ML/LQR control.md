# Summary 

**LQR (Linear Quadratic Regulator) control** is a powerful and widely used **optimal control method** for systems that can be described by linear dynamics.

- State feedback control method for [[linear dynamic systems]].
- Select the gain of the [[linear controller]] such that the [[quadratic cost function]] is minimized.
