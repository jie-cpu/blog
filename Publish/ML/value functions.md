# Summary 

**value functions** help an agent understand how good a state or action is based on expected future rewards.
1. **Purpose**: They show the long-term benefit of being in a certain state or taking a specific action.
2. **Types**:
    - **State Value Function (V(s))**: Tells how much reward to expect from being in state s.
    - **Action Value Function (Q(s,a))**: Tells how much reward to expect from taking action a in state s.
3. **Guidance**: Value functions help the agent decide which states or actions are better.
