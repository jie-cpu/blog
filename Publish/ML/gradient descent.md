# Summary 

**Gradient Descent** is a fundamental **optimization algorithm** used in machine learning, especially for training neural networks.

Its core purpose is to **find the set of parameters (weights and biases) for a model that minimize the cost/loss function**.

- **Iterative Adjustment:** It works by iteratively adjusting the model's parameters in the direction that reduces the cost function most steeply.
- **"Downhill Walking":** Imagine being on a mountain (the cost function surface) and wanting to reach the lowest point (the minimum cost). Gradient Descent is like taking small steps downhill.
- **Gradient:** The "gradient" tells us the direction of the steepest ascent (uphill). Gradient Descent moves in the _opposite_ direction (steepest descent).
- **Learning Rate:** Each "step" taken is controlled by a parameter called the "[[learning rate]]," which determines how big or small each adjustment to the parameters will be.
In essence, Gradient Descent is the engine that drives machine learning models to learn. It systematically tweaks the model's internal settings to make its predictions increasingly accurate by always moving towards the lowest possible error.
