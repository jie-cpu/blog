# Summary 

A **Recurrent Neural Network (RNN)** is a type of neural network specifically designed to **process sequential data**, where the order of information matters.

Its core purpose is to **handle inputs that are sequences (like text, speech, or time series data) by maintaining an internal "memory" of previous elements in the sequence.**

- **Recurrent Connections:** Unlike traditional neural networks where information flows in one direction, RNNs have loops or "recurrent" connections. This allows information from a previous step in the sequence to influence the current step's output.
- **"Memory":** This internal memory (often called a "hidden state") allows RNNs to understand context and dependencies across different points in a sequence.
- **Applications:** Highly effective for tasks involving sequences, such as:
    - **Natural Language Processing ([[NLP]]):** Language translation, text generation, sentiment analysis.
    - **[[Speech Recognition]]:** Converting audio to text.
    - **[[Time Series Prediction]]:** Forecasting stock prices, weather.
In essence, an RNN is like a neural network with a short-term memory. It can "remember" what happened earlier in a sequence to help it understand and process the current information, making it ideal for tasks where context over time is crucial.
