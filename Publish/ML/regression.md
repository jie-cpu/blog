# Summary 
**Regression** is a fundamental task in machine learning, also falling under the umbrella of **supervised learning**.

Its core purpose is to **predict a continuous numerical value** for a given input.

- **Continuous Output:** Unlike classification (which predicts categories), regression models output a number that can fall anywhere within a range (e.g., a decimal value, not just a fixed label).
- **Training with Numerical Labels:** The model learns to predict by being trained on a dataset where each input example is associated with a correct, continuous numerical value (e.g., house features paired with house prices, or historical data paired with future temperatures).
- **Estimating Values:** Once trained, the model can then take new, unseen input data and estimate its corresponding numerical value.
In essence, regression is like drawing a line or curve through data points to predict where a new point would fall. For example, a house price regression model estimates the exact price of a house based on its features.
