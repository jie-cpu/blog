# Summary 

**Linear Regression** is one of the simplest and most fundamental algorithms in **supervised learning**, used for **regression tasks**.

Its core purpose is to **find a linear relationship between input features and a continuous target variable**, allowing for prediction.

- **Linear Relationship:** It assumes that the output (the value we want to predict) can be directly modeled as a straight line (or a flat hyperplane in higher dimensions) based on the input features.
- **Predicting Continuous Values:** It predicts a continuous numerical output (e.g., house price, temperature), unlike classification which predicts categories.
- **Best-Fit Line:** The algorithm works by finding the "best-fitting" straight line (or plane) through the training data that minimizes the distance (often measured by Mean Squared Error) between the line and the actual data points.
In essence, Linear Regression is like trying to draw the best straight line through a set of scattered data points to predict new values. For example, predicting a student's test score based on the number of hours they studied, assuming a direct, linear relationship.
