# Summary 
**Classification** is a fundamental task in machine learning, falling under the umbrella of **supervised learning**.

Its core purpose is to **categorize input data into one of several predefined classes or categories**.

- **Categorical Output:** The model predicts a discrete, categorical label for each input.
- **Training with Labels:** A model learns to classify by being trained on a dataset where each input example is already assigned its correct category (e.g., "spam" or "not spam," "cat" or "dog," "fraudulent" or "legitimate").
- **Predicting Groups:** Once trained, the model can then take new, unseen data and assign it to the most probable class it has learned.
In essence, classification is like sorting items into predefined bins based on their characteristics. For example, an email classifier sorts emails into "inbox" or "spam" based on its content.
