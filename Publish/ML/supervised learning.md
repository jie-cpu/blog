# Summary 

**Supervised Learning** is a type of machine learning where the model learns by example, from a dataset of **labeled data**.

Its core purpose is to **learn a mapping function from input data to an output label**, by observing many examples of both.

- **Labeled Data:** The dataset consists of input examples, each paired with the correct or "ground truth" output (the "label").
- **Learning from Examples:** The model is "supervised" in its learning process, meaning it constantly compares its predictions with the provided correct labels and adjusts itself to minimize the difference.
- **Prediction:** Once trained, the model can then predict the labels for new, unseen input data.
In essence, supervised learning is like teaching a child by showing them many examples (e.g., "this is an apple," "this is a banana") until they can correctly identify new, unseen objects.
