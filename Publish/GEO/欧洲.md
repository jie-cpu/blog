## 九、欧洲语系总结

| 语系 | 代表国家 | 特点 |
| ---- | -------- | ---- |
| **印欧语系 - 日耳曼语族** | 英国、德国、荷兰、瑞典、挪威 | 欧洲最大语族之一，分西、北、东三支 |
| **印欧语系 - 罗曼语族** | 法国、意大利、西班牙、葡萄牙、罗马尼亚 | 源自拉丁语，分布在南欧和西欧 |
| **印欧语系 - 斯拉夫语族** | 俄罗斯、波兰、捷克、保加利亚、塞尔维亚 | 分东、西、南三支，分布于东欧和中欧 |
| **印欧语系 - 凯尔特语族** | 爱尔兰（少数地区） | 曾遍布西欧，现在主要在英国周边残存 |
| **印欧语系 - 希腊语族** | 希腊 | 独立分支，3000多年历史 |
| **印欧语系 - 阿尔巴尼亚语族** | 阿尔巴尼亚 | 独立分支，与希腊语和斯拉夫语关系较远 |
| **印欧语系 - 波罗的语族** | 拉脱维亚、立陶宛 | 现存最古老的印欧语系分支之一 |
| **印欧语系 - 亚美尼亚语族** | 亚美尼亚 | 独立分支，有独特的字母系统 |
| **乌拉尔语系** | 芬兰、匈牙利、爱沙尼亚 | **非印欧语系**，与土耳其语无亲缘关系；源自乌拉尔山脉地区 |
| **突厥语系** | 阿塞拜疆 | 与土耳其语、哈萨克语等同源 |
| **南高加索语系** | 格鲁吉亚 | **完全独立语系**，与任何其他语系无关 |
| **亚非语系 - 闪米特语族** | 马耳他 | 欧洲唯一亚非语系国家，与阿拉伯语和希伯来语同源 |
