## 五、语系分布总结

| 语系 | 代表国家 | 分布区域 | 特点 |
| ---- | -------- | -------- | ---- |
| **印欧语系 - 罗曼语族** | 墨西哥、巴西、阿根廷、哥伦比亚、秘鲁 | 拉丁美洲（几乎所有国家） | 源自拉丁语：**西班牙语**（18国）、**葡萄牙语**（巴西）、**法语**（海地、法属圭亚那） |
| **印欧语系 - 日耳曼语族** | 美国、加拿大、牙买加、圭亚那 | 北美洲、加勒比、南美洲北部 | **英语**为主，加拿大也使用**法语** |
| **孤立语言 / 土著语言** | 巴拉圭（瓜拉尼语） | 南美洲 | 瓜拉尼语为巴拉圭官方语言之一，与西班牙语并列 |
| **南岛语系** | — | — | 美洲大陆没有南岛语系国家 |

## 📌 补充说明

| 项目 | 说明 |
| ---- | ---- |
| **拉丁美洲定义** | 指以罗曼语族语言（西班牙语、葡萄牙语、法语）为官方语言的美洲国家，包括墨西哥、中美洲、加勒比及南美洲（除美国和加拿大） |
| **北美洲 vs 拉丁美洲** | 北美洲通常只指加拿大、美国、墨西哥（3国）；中美洲和加勒比有时被归入"拉丁美洲"范畴 |
| **官方语言特点** | 南美洲除巴西（葡萄牙语）、圭亚那（英语）、苏里南（荷兰语）外，均为西班牙语国家 |
| **法属圭亚那** | 不是独立国家，而是法国的一个海外省，属于欧盟的一部分 |
| **非主权地区** | 格陵兰（丹麦）、波多黎各（美国）、百慕大（英国）等未列入本表 |
