## 🧠 词源彩蛋

| 单词/片名 | 有意思的点 |
| --------- | ---------- |
| **Memento** | 拉丁语 *mementō* 是 *meminisse*（记住）的命令式，意为“**记住！**”——正好契合电影关于记忆的主题 |
| **Inception** | 原意是“**开始**”，但电影里特指“在某人脑海中植入一个想法”（*inception of an idea*） |
| **Tenet** | 这是一个**回文词**（palindrome），正着读反着读都一样，呼应电影中时间倒流的设定 |
| **Prestige** | 魔术术语中的最后一个阶段 —— “**化境**”（把消失的东西变回来），用的正是“声望/奇迹”的双关 |
| **Doodlebug** | 一词多义：①昆虫“蚁蛉”；②二战德国V-1导弹绰号；③小型侦察车。短片内容是一个男人追打小虫，实则是在打缩小的自己——片名本身就是精妙隐喻 |
| **Larceny** | 法律英语专业术语，比*theft*更正式，诺兰早期短片多用这种简洁有力的一词标题 |
| **The Odyssey** | 荷马史诗之一，讲述**奥德修斯**在特洛伊战争后历经十年漂泊重返家乡的故事 |
