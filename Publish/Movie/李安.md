## 🧠 有趣的词源彩蛋

| 单词 / 片名 | 有意思的点 |
| ----------- | ---------- |
| **Eat Drink Man Woman** | 英文片名对中文"饮食男女"逐字直译，简洁有力。源自《礼记》："饮食男女，人之大欲存焉"，指**食欲与性欲是人的基本欲望**。法语译名"盐和糖"则更有意境的引申意味。 |
| **Brokeback** | *Brokeback* 由 *broke*（折断的）+ *back*（背部/山脊）构成，字面指 **"断裂的山脊"** 。直译为《断背山》准确，而《断臂山》译法则融合了中文"断臂"与"手足情深"的文化联想。 |
| **Gemini Man** | *Gemini* 在拉丁语中意为 **"双胞胎"** ，也是黄道"双子座"的名称。片名一语双关：既指"双子座"的性格特质，又暗合"一个男人遭遇自己的年轻克隆体"（如双胞胎）的剧情核心。 |
| **Crouching Tiger, Hidden Dragon** | 英文对"卧虎藏龙"逐字直译，保留了中国武侠特有的**意象感**——"蹲伏的虎，隐藏的龙"，虽然直白却极有画面感。 |

## 📝 总结

李安电影的英文片名大致分为三类：
1. **直译型**（*Pushing Hands*、*Eat Drink Man Woman*、*Crouching Tiger, Hidden Dragon*）：保留中文原意，甚至逐字对应，虽有时显得"笨拙"却独具辨识度。
2. **意译型**（*Sense and Sensibility*、*The Ice Storm*）：完全采用英文语境下的命名逻辑，与原片名对应但各有表达。
3. **双关型**（*Brokeback Mountain*、*Gemini Man*）：英文原词本身就暗含多重含义，与影片主题形成巧妙的呼应。
