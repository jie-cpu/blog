## 2️⃣ 政治 / 社会 (Politics & Society)

| 英文 | 中文翻译 | 说明 |
|------|----------|------|
| **National anthem** | 国歌 | 代表国家的正式歌曲 |
| **Propaganda** | 宣传 | 多指带有特定政治目的的宣传 |
| **Government propaganda** | 政府宣传 | 由政府主导的宣传 |
| **Political propaganda** | 政治宣传 | 政党或政治运动进行的宣传 |
| **Scandal** | 丑闻 | 引起公众负面反响的事件 |
| **Exposé** | 揭露 / 曝光 | 揭露丑闻或不法行为的报道 |
| **Start genocides** | 发动种族灭绝 | 系统性消灭某一族群 |
| **Civilians** | 平民 | 非军事人员 |
| **Ethnic group** | 族群 | 具有共同文化、语言或血统的群体 |
| **Ethnic minority** | 少数民族 | 在人口中占少数的族群 |
| **Positive racism** | 正向种族主义 / 善意种族主义 | 以"正面"刻板印象看待某族群，实则仍是偏见 |

## 3️⃣ 计算机 / 网络安全 (Computer & Cybersecurity)

| 英文                               | 中文翻译    | 说明                 |
| -------------------------------- | ------- | ------------------ |
| **Red hat**                      | 红帽黑客    | 类似白帽，但攻击手段更激进      |
| **White hat**                    | 白帽黑客    | 道德黑客，合法测试安全漏洞      |
| **Black hat**                    | 黑帽黑客    | 恶意黑客，非法入侵系统        |
| **Gray hat**                     | 灰帽黑客    | 介于白帽与黑帽之间，有时违法但无恶意 |
| **Web crawler / Spider**         | 网络爬虫    | 自动抓取网页信息的程序        |
| **Web harvesting**               | 网络数据采集  | 从网页中提取数据的自动化过程     |
| **Scrape the data from the web** | 从网页抓取数据 | 爬取网页内容的操作          |
